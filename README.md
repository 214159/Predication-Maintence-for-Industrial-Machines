failure type across 6 categories, and estimates the **Remaining Useful Life (RUL)** — all through
an interactive Streamlit dashboard with live sensor inputs.

## 🧠 How It Works

```
Sensor Data ──► Feature Engineering ──► XGBoost Classifier ──► Failure Type
                                                │
                                                └──► XGBoost Regressor ──► RUL (minutes)
```

**Stage 1 — Classification:** XGBoost detects which of 6 failure modes is most likely.

**Stage 2 — Regression:** If a failure is detected, estimates minutes of useful life remaining
from cumulative tool wear.

**Explainability:** SHAP `TreeExplainer` waterfall plots show exactly which sensor drove the risk.

## 🔬 Dataset
**[AI4I 2020 Predictive Maintenance Dataset](https://www.kaggle.com/datasets/abdelazizsami/predictive-maintenance-dataset)** — Kaggle

| Property | Value |
|---|---|
| Records | 10,000 |
| Failure Rate | ~3.4% (class imbalanced) |
| Input Features | 8 |
| Failure Classes | 6 |

### Input Features

| Feature | Description |
|---|---|
| `Type` | Machine quality grade — H / M / L |
| `Air temperature [K]` | Ambient air temperature |
| `Process temperature [K]` | Tool process temperature |
| `Rotational speed [rpm]` | Tool rotational speed (log-transformed) |
| `Torque [Nm]` | Applied rotational force |
| `Tool wear [min]` | Cumulative tool usage time |

### Failure Classes

| Class | Description |
|---|---|
| `No_Failure` | Machine operating normally |
| `TWF` | Tool Wear Failure |
| `HDF` | Heat Dissipation Failure |
| `PWF` | Power Failure |
| `OSF` | Overstrain Failure |
| `RNF` | Random Failure |
## 🤖 Model Details

### XGBoost Classifier

```python
XGBClassifier(
    n_estimators=500,
    learning_rate=0.03,
    max_depth=6,
    subsample=0.9,
    colsample_bytree=0.9,
    gamma=0.1,
    min_child_weight=3,
    random_state=42
)
```

- StratifiedKFold (5-fold) cross-validation
- Prediction threshold tuned to **0.6** to boost failure recall
- Class imbalance handled via stratified train/test split

### XGBoost Regressor

```python
XGBRegressor(
    n_estimators=200,
    max_depth=8,
    learning_rate=0.05,
    objective='reg:squarederror'
)
```

- Trained only on machines with detected failures
- RUL target = `max(Tool_wear_min) - Tool_wear_min`

## ⚡ Quick Start

### 1. Clone the repo

```bash
git clone https://github.com/your-username/predictive-maintenance.git
cd predictive-maintenance
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Train the models

Run all cells in `predictive_maintenance.ipynb`. The final cell saves:

```
classifier_model.pkl
regressor_model.pkl
label_encoder.pkl
```

### 4. Launch the dashboard

```bash
streamlit run predictive_maintenance_app.py
```

Open `http://localhost:8501` in your browser.
